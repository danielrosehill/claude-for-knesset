# Platform: Program for Government

This folder holds the **meta-proposal** — the rolled-up program for government synthesized from the completed policy papers in [`policy/`](../policy/).

Nothing enters the program until its policy area has passed stage 3 of the [pipeline](../PIPELINE.md): a compiled paper with sourced evidence, costed mechanisms, and a steelmanned trade-offs section. The program is the distillation, not the workspace.

`program-for-government.md` will be created by `/update-platform` once the first paper lands.
